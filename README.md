# Gottp Http Client
Gottp is a wrapper over the base http's client functionality written in go, providing additional features of tracing and resilience

## Installation
1. Install module
```sh
go get github.com/betalixt/gottp
```
2. Import
```go
import "github.com/betalixt/gottp" 
```
